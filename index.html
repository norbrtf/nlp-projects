Implementacja i ewaluacja systemu RAG
Zespół: 2-3 osoby | Język: Python

Implementacja i ewaluacja systemu RAG (Retrieval-Augmented Generation) dla specjalistycznej dziedziny wiedzy dziedzinowej.

Zadania:

  • Stworzyć korpus dokumentów specjalistycznych w wybranej dziedzinie (np. inżynieria, IT, programowanie itp.)
  • Zaprojektować architekturę i zaimplementować system RAG wykorzystujący wybraną metodę indeksowania i wyszukiwania (np. dense retrieval z embedding, sparse retrieval) wraz z jej argumentacją
  • Porównać skuteczność różnych modeli LLM (self-hosted lub/i poprzez API) jako komponentów generujących odpowiedzi w systemie RAG (np. porównanie 2 modeli)
  • Przeprowadzić ewaluację systemu pod kątem dokładności odpowiedzi, halucynacji oraz efektywności wyszukiwania, kontekstowości z wykorzystaniem metryk np. RAGAS lub innej wybranej biblioteki
  • Zbadać i zaproponować techniki optymalizacji systemu (np. Self-RAG, Corrective-RAG, Adaptive RAG lub inne)

Context Engineering - zaawansowane zarządzanie kontekstem
Zespół: 2-3 osoby | Język: Python

Implementacja i ewaluacja technik zaawansowanego zarządzania kontekstem (context engineering) w systemach agentowych/RAG.

Zadania:

  • Zbadać i zaimplementować system konwersacyjny (np. czat) obsługujący zarządzanie długim kontekstem (np. kilkunasto/kilkudziesięcio stronicowe dokumenty, >10 tys. tokenów per dokument) z wybranymi technikami np. semantic chunking, document structure-based chunking w wybranej dziedzinie (np. inżynieria, IT, programowanie itp.)
  • Zaimplementować wybraną metodę query understanding np. query rewriter
  • Porównać co najmniej 2 strategie context engineering np. context compression/summarizing, hierarchical summarization
  • Zbadanie oraz zaimplementowanie wybranej metody context cache/semantic cache
  • Zbadanie problemu "Lost-in-the-middle" i sposobów jego mitygacji
